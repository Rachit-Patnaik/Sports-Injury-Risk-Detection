# Sports Injury Risk Detection

An AI-powered application that analyzes athlete videos to detect biomechanical issues, estimate injury risk, and recommend corrective exercises using Computer Vision, Pose Estimation, and Machine Learning.

## Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
- Pandas
- Matplotlib
